# BBEdit syntax highlighting

## TLDR
Use &mdash; character instead of "-" character in HTML comments

### This is how it looks
    <!--
    // ———————————————————————————————————————————————————————————————————
    // White block
    // ———————————————————————————————————————————————————————————————————
    // - Slider
    // ———————————————————————————————————————————————————————————————————
    -->
